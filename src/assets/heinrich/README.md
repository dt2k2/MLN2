# Heinrich hero loops

Drop the seven silent H.264 MP4 files in this directory. Vite discovers them
automatically; no component changes are required.

| File                    | UI state      | Loop direction                                                      |
| ----------------------- | ------------- | ------------------------------------------------------------------- |
| `01-neutral.mp4`        | Điềm tĩnh     | Heinrich reads the ledger; machinery runs steadily.                 |
| `02-expansion.mp4`      | Tham vọng     | He studies machinery plans and looks up with restrained confidence. |
| `03-hardline.mp4`       | Cứng rắn      | Tense posture, a glance at the clock, faster machinery behind him.  |
| `04-market-crisis.mp4`  | Lo âu         | Full warehouse or loss ledger, colder light and slower machinery.   |
| `05-labor-conflict.mp4` | Bị thách thức | Workers gather beyond the window; no explicit violence.             |
| `06-rupture.mp4`        | Mất kiểm soát | Loose papers, dense smoke and intermittent red light.               |
| `07-dominant.mp4`       | Đắc thế       | Strong factory output and positive accounts, without triumphalism.  |

Production specification: 960x540, 16:9, 24 fps, 4-6 seconds, H.264 with
`yuv420p`, no audio, text or fade-to-black. Keep Heinrich and the camera fixed
across all clips, with his face and torso inside the center 60% safe area.
Target 0.8-1.5 MB per file and a seamless first/last frame.
